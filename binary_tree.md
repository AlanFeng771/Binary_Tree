# Binary_Tree
實現Binary tree 的各種 func

## Description

了解並實作資料結構的各種結構範例
## Getting Started

### Dependencies

* g++/gcc

### Installing

* main.cpp 是程式的進入點，如果你想要測試必須要引入欲測試的檔案在main.cpp 的最上方。
* ex:
``` cpp=
#include "Binary_Tree.h"
```

### Executing program

``` 
g++ main.cpp Binary_Tree.cpp
```
